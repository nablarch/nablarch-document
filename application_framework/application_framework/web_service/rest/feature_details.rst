.. important::

  RESTfulウェブサービスでは `ETag` や `If-Match` を使用した楽観的ロックには対応していない。
  そのため、RESTfulウェブサービスで楽観的ロックを行う際は、リクエストボディに直接バージョン番号を含めること。

.. important::

  :ref:`exclusive_control` 機能は、クライアント(taglib)との連動が前提であるため、
  RESTfulウェブサービスでは使用できない。
